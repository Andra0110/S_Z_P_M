# Minimum Viable Product (MVP)

## Goal

Create a console-based clinic management system that supports basic management of patients, doctors and appointments.

---

## Patient Management

The system shall allow:

- add patient
- remove patient
- search patient
- display patient list

---

## Doctor Management

The system shall allow:

- add doctor
- assign specialization
- display doctor list

---

## Appointment Management

The system shall allow:

- create appointment
- cancel appointment
- display appointments

---

## Data Persistence

The system shall allow:

- save data to file
- load data from file

---

## User Interface

The system shall provide:

- main menu
- patient menu
- doctor menu
- appointment menu

---

## Out of Scope

The following features are not included in MVP:

- graphical user interface
- database integration
- multi-user support
- authentication system
- online appointment booking
